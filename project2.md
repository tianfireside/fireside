# Project 2 — Events Page

## What we're building
An events page on fireside.law matching the site's existing design, with a simple JSON file in the repo for weekly event updates. Push to GitHub → Vercel auto-deploys.

## Each event includes
- Title
- Date
- Time
- Location
- Description
- RSVP link (mailto or Google Form)

## Deliberately out of scope (for now)
- Email collection
- Automated reminders
- Waitlists

1. What to call it in the nav — "Events", "What's On", something else?
2. A real upcoming event to use as sample data
